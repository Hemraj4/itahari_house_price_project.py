Itahari House Price Prediction & Market Insight System
Files created:
- itahari_house_prices_large.csv  (3500 synthetic rows)
- itahari_house_price_project.py  (project script template)
How to use:
1. Download the CSV and the project script.
2. Place both files in the same folder on your machine.
3. Create a virtual environment and install dependencies:
   pip install pandas numpy matplotlib scikit-learn joblib
4. Run the project script:
   python itahari_house_price_project.py
Notes:
- The project script trains RandomForest and GradientBoosting models and saves results under 'models/' and 'figures/'.
- You can convert the script into a Jupyter notebook for step-by-step exploration.
